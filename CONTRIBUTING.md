# Contributing to MaskBeforeAI

Thank you for your interest in MaskBeforeAI!

## Reporting Issues

We welcome bug reports and feature requests via [GitHub Issues](https://github.com/kisumgmbh/mask-before-ai/issues).

When reporting a bug, please include:

- **Browser name and version** (e.g., Chrome 124, Edge 124)
- **AI platform** where the issue occurred (e.g., ChatGPT, Claude)
- **Steps to reproduce** the issue
- **Expected behavior** vs. what actually happened
- **Screenshots** if applicable

## Feature Requests

Have an idea for a new feature? [Open an issue](https://github.com/kisumgmbh/mask-before-ai/issues) and describe:

- What you'd like to see
- Why it would be useful
- Any examples or references

## Source Code

MaskBeforeAI is a proprietary product. The source code is not included in this repository. This repo serves as the public-facing project page for issue tracking, feature requests, and community discussion.
