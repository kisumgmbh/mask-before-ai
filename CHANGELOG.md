# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-03-07

### Added
- Pro tier: PDF, DOCX, code, and text file masking with multi-page review
- PDF export for masked documents
- Promo video on landing page
- Error reporting API

### Changed
- Redesigned landing page with gallery demos and SVG animations

## [0.1.0] - 2026-02-23

### Added
- Browser extension for Chrome/Edge (Manifest V3)
- Manual masking: draw boxes and brush strokes
- Three redaction modes: blur, pixelate, black box
- Support for 9 AI platforms (ChatGPT, Claude, Copilot, Gemini, Perplexity, DeepSeek, Grok, Meta AI, Mistral)
- Interactive review dialog with undo/redo
- Keyboard shortcuts for review dialog tools
- Client-side Canvas redaction engine (no server required)
- Settings page with extension toggle, intercept controls, and mask style selection
- Enterprise feature preview with contact dialog
- Landing page at mask.ki-sum.ai
- Apache 2.0 license
